# ADR 0001: Record architecture decisions

- Status: Accepted
- Date: 2026-06-26

## Context
Architecturally significant decisions must be tracked (CLAUDE.md §23).

## Decision
We use lightweight ADRs in `docs/adr/`. Any change to the CLAUDE.md pillars, the Tool
contract, the agent roster, the Framework Engine model, or the Mission Lifecycle requires
an ADR and an update to CLAUDE.md.

## Consequences
A durable, reviewable history of why the architecture is the way it is.
