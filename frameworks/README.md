# Frameworks (as data)

Compliance frameworks live here as **data, not code** (CLAUDE.md §13). Adding a framework
or a new version is a PR that touches only this folder. Supported: NCA ECC, SAMA, PDPL,
ISO 27001, NIST CSF, CIS, COBIT, COSO, and any future framework.

To add one, see `docs/architecture/PROJECT_SKELETON.md` §14.3.
